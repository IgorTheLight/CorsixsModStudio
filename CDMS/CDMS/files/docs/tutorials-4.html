Using LUAs instead of RGDs

Many people prefer to edit unit stats as LUA files rather than RGD files, as it is easier to copy & paste sections, search for certain text and you can take advantage of inheritance.

RGD Macro Basics

Want to make a similar change to a large number of RGD files at once? No longer is it a boring and repetitive task. A small macro can make all the changes for you in the blink of an eyelid.

BFX Editor

BFX files can be found in data\ art\ fx. At the file format level, they are the same as RGD files, so the editor for BFX files is the RGD editor. They are fairly complex though; not at all like normal RGD files. This is all that is known about BFX files - if you have more information, please submit it.
